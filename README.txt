- Descargar Node.js la version recomendada a traves de la pagina https://nodejs.org/es/
- Instalar el ejecutable descargado
- Revisar a traves de CMD si la instalacion fue correcta a traves del siguiente comando : node -v
- Instalar NPM a traves de CMD con el siguiente comando: npm install npm@latest -g
- Ejecutar dentro de la carpeta raiz del proyecto en CMD el siguiente comando: npm install
- Luego, ejecutar dentro de la carpeta raiz del proyecto en CMD el siguiente comando: ng serve
- Ingresar esta url en el navegador : http://localhost:4200/
- Enjoy...
